# GESTOR ACADÉMICO DE COLEGIO SAN JOSE 🎓

![C#](https://img.shields.io/badge/C%23-239120?style=for-the-badge&logo=c-sharp&logoColor=white)
![.NET](https://img.shields.io/badge/.NET-512BD4?style=for-the-badge&logo=dotnet&logoColor=white)
![SQL Server](https://img.shields.io/badge/SQL%20Server-CC2927?style=for-the-badge&logo=microsoft-sql-server&logoColor=white)

Es una aplicación web diseñada para la gestión integral de institucion educativa. Permite administrar de manera eficiente estudiantes, docentes, materias y el rendimiento académico general.

## ✨ Características Principales

* **Dashboard Inteligente:** Visualización rápida de métricas clave (estudiantes totales, docentes, materias).
* **Gestión de Estudiantes:** Registro completo de expedientes, contacto y niveles académicos.
* **Control de Materias:** Organización del catálogo de asignaturas.
* **Sistema de Calificaciones:** Registro y seguimiento del rendimiento escolar.

## 🛠️ Requisitos del Sistema

* **SO:** Windows 10 o superior.
* **IDE:** Visual Studio 2022.
* **Framework:** .NET Framework 4.7.2 / .NET 6.0+.
* **Base de Datos:** SQL Server (Script incluido en `BD SQL`).

## 🚀 Instalación y Configuración

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/henrychacon914-design/COLEGIO-SAN-JOSE
    ```
2.  **Configurar la Base de Datos:**
    * Abre SQL Server Management Studio (SSMS).
    * Ejecuta el script ubicado en la carpeta `BD SQL/BD.sql`.
3.  **Abrir el Proyecto:**
    * Abre el archivo `colegio san jose` en Visual Studio.
    * Restaura los paquetes NuGet (clic derecho en la Solución > Restaurar paquetes NuGet).
4.  **Ejecutar:**
    * Presiona `F5` o haz clic en el botón **Iniciar**.

## 📂 Estructura del Proyecto

* `/Forms`: Interfaces de usuario (Windows Forms).
* `/Classes`: Lógica de negocio y modelos de datos.
* `/Data`: Capa de acceso a datos y conexión SQL.
* `/Resources`: Iconos, imágenes y assets del sistema.
* `/Database`: Script `.sql` para la creación de la base de datos.
